## Architecture

- `connector/` — Apps Script, Cloudflare Worker, and OpenAPI
- `ghost-theme/` — production Ghost theme source
- `homepage/` — standalone visual prototype
- `content/` — canonical content
- `knowledge/` — taxonomy and knowledge structures
- `automation/` — build and release scripts
- `docs/` — operations and release documentation
- `releases/` — generated release artifacts
