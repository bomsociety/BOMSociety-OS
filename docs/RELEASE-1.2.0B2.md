# BOMSociety OS 1.2.0B2

## Hero improvements
- Kept the permanent three-line mission as the hero.
- Assigned distinct, high-contrast colors to each line.
- Enlarged “Improve your decisions.” as the principal action-oriented line.
- Increased the product-definition subheadline.
- Increased and separated the 30-second / 2-minute / 5-minute statements.
- Reduced top and internal spacing so the principal content fits above the fold more reliably.
- Reduced the graph height while preserving the animated visual.
- Mirrored all changes in the local prototype and Ghost theme.

## Design basis
Color is used to create hierarchy rather than decoration. The palette preserves high text/background contrast and does not use color as the sole carrier of meaning.
